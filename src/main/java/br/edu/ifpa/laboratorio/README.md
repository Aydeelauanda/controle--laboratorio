# Sistema de Controle de Empréstimo de Equipamentos

## Integrantes
- Aydêe Lauanda Costa Barroso

## Descrição do problema
O sistema foi desenvolvido para controlar empréstimos de equipamentos em laboratório, permitindo cadastro de alunos, cadastro de equipamentos, realização de empréstimos e devoluções.

## Requisitos implementados
- Cadastro de alunos
- Cadastro de equipamentos
- Listagem de equipamentos disponíveis
- Realização de empréstimos
- Bloqueio de equipamentos indisponíveis
- Registro de devolução
- Histórico de empréstimos
